# GoHugo代码笔记

hugo是一个很NB的项目。 

[官网](http://gohugo.io/)

Hugo可以干啥：

- 生成静态网站
- 速度快，灰常灰常快
- 模板引擎
- shortcode模型
- 100+ 优美主题套件
- 支持 Markdown/Mmark/`Emacs Org-Mode`/RST/Asciidoc/Pandoc 输入
    - Markdown/`Emacs Org-Mode` 内建，其它通过集成工具完成
- 支持 html/json/RSS 输出

[Hugo Features](https://gohugo.io/about/features/)
